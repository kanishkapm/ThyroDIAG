Project Summary

Gene Expression Threshold

When combining multiple GEO datasets, different datasets may have different expression values for the same gene due to batch effects or different normalization methods.

Mean Expression Calculation:
Mean(gene) = Σ(expression_value) / n_samples

Example: PAX8 = (1.45 + 1.52 + 1.56 + ...) / 360 = 1.5047

Random Forest

Type: Ensemble (Bagging)

Description: An ensemble of decision trees using bootstrap aggregating (bagging). Each tree is trained on a random subset of features.

Key Parameters:
n_estimators = 100
max_depth = 10
random_state = 42

How it works:

  1. Create bootstrap samples from training data
  2. Train decision trees on each sample
  3. Each tree makes a prediction
  4. Final prediction = majority vote

Support Vector Machine (SVM)

Type: Kernel-based

Description: Finds the optimal hyperplane that maximizes the margin between different classes in feature space.

Key Parameters:
kernel = 'rbf'
probability = True
random_state = 42

How it works:

  1. Map data to higher-dimensional space
  2. Find optimal separating hyperplane
  3. Use kernel functions for non-linear separation
  4. RBF kernel handles complex boundaries

Gradient Boosting

Type: Ensemble (Boosting)

Description: Sequential ensemble method that builds trees to correct errors of previous trees.

Key Parameters:
learning_rate = 0.1
n_estimators = 100
max_depth = 3

How it works:

  1. Initialize with simple model
  2. Calculate residuals (errors)
  3. Fit new tree to residuals
  4. Update model: F(x) = F(x-1) + η * h(x)

K-Nearest Neighbors (KNN)

Type: Instance-based

Description: Classifies samples based on the majority class of their k nearest neighbors in feature space.

Key Parameters:
n_neighbors = 5
metric = 'minkowski'
weights = 'distance'

How it works:

  1. Calculate distance to all training points
  2. Find k nearest neighbors
  3. Vote on class labels
  4. Assign majority class

Multi-Layer Perceptron (MLP)

Type: Neural Network

Description: A feedforward neural network with multiple hidden layers that learns non-linear relationships.

Key Parameters:
hidden_layer_sizes = (100, 50)
activation = 'relu'
max_iter = 1000

How it works:

  1. Input → Hidden layers → Output
  2. Each neuron applies activation function
  3. Backpropagation for weight updates
  4. Softmax for multi-class output

Stacking Ensemble

Type: Meta-Ensemble

Description: Two-level ensemble where base learners' predictions become features for a meta-learner.

Base Models:
- LASSO (LogisticRegressionCV)
- ElasticNet
- Random Forest
- SVM

Meta-Learner:
- Logistic Regression

How it works:

  1. Train base models with cross-validation
  2. Collect out-of-fold predictions
  3. Create meta-features
  4. Train meta-learner on meta-features

Hybrid Ensemble (Deep Learning + Traditional ML)

Combines the strengths of both deep learning and traditional machine learning approaches:

  1. Deep Learning Models: 1D CNN, LSTM, Transformer
  2. Traditional Models: Random Forest, SVM, Gradient Boosting
  3. Weighted Voting: Based on cross-validation performance
  4. Probability Averaging: Mean of all model probabilities
Ensemble Prediction:
P(final) = Σ(w_i * P_i) / Σ(w_i)

where w_i = CV accuracy of model i

Key Concepts Explained

Cross-Validation

A technique to assess how well a model generalizes to independent datasets. The data is split into k folds, and the model is trained k times, each time using k-1 folds for training and 1 fold for validation.

ROC-AUC

Receiver Operating Characteristic - Area Under Curve. Measures the model's ability to distinguish between classes. A value of 1.0 means perfect classification, 0.5 means random guessing.

LASSO Regression

Least Absolute Shrinkage and Selection Operator. Adds L1 regularization that can shrink some coefficients to exactly zero, performing feature selection automatically.

Regularization

Techniques to prevent overfitting by adding a penalty term to the loss function. L1 (LASSO) promotes sparsity, L2 (Ridge) promotes small weights.

Batch Normalization

Normalizes the inputs of each layer to have zero mean and unit variance. This stabilizes training and allows for higher learning rates.

Dropout

Randomly sets a fraction of input units to 0 during training. This prevents co-adaptation of neurons and acts as regularization.

Evaluation Metrics

Metric Formula Description
Accuracy (TP + TN) / (TP + TN + FP + FN) Overall correctness of predictions
Precision TP / (TP + FP) Of predicted positives, how many are correct
Recall TP / (TP + FN) Of actual positives, how many are predicted
F1-Score 2 × (Precision × Recall) / (Precision + Recall) Harmonic mean of precision and recall
ROC-AUC Area under ROC curve Model's ability to rank positive samples higher
