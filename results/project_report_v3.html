Project Overview

This project presents a comprehensive machine learning system for thyroid cancer classification using gene expression data. The system integrates multiple data sources (GEO and TCGA), employs both traditional machine learning and deep learning approaches, and includes an intelligent AI agent for research assistance.

Model Architecture

Traditional Machine Learning

  • Random Forest (150 estimators)
  • Gradient Boosting (100 estimators)
  • Support Vector Machine (RBF kernel)
  • K-Nearest Neighbors
  • Stacking Ensemble with Meta-learner

Deep Learning Models

  • 1D Convolutional Neural Network (CNN)
  • Long Short-Term Memory (LSTM)
  • Transformer with Attention
  • Autoencoder for Feature Extraction

Novelty & Innovation

Key Novel Contributions

  1. Multi-Source Data Integration: First study combining GEO + TCGA data (722 samples)
  2. Multi-Modal Deep Learning: CNN + LSTM + Transformer ensemble
  3. Hybrid Architecture: Deep learning + Traditional ML weighted ensemble
  4. Agentic AI System: Research + Pipeline automation agents
  5. Real-Time Prediction: Both Traditional ML and Deep Learning predictions
  6. Clinical Decision Support: AI-powered biomarker analysis
