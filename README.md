# Thyroid Cancer Biomarker Prediction Pipeline

This project implements a machine learning and deep learning pipeline for thyroid cancer classification using gene expression datasets (GEO & TCGA). The workflow includes preprocessing, feature selection (LASSO), ensemble modeling, pathway analysis and visualization.

Try this in online : https://thyrodiag.onrender.com/

Developed as part of M.Sc Bioinformatics research work
